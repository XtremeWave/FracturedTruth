## 关于

FinalSuspect是一个适用于Among Us的原版优化插件。\
插件为原版带来了10+项优化,使得原版也能获得更好的体验。\
有关该插件的详细使用文档(安装,更新以及完整的配置说明),请参阅[网站文档](https://www.xtreme.net.cn/docs/FS/)。
