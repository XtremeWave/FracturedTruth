# 安全政策

## 受支持的FinalSuspect版本

目前,XtremeWave支持以下版本,其中:
- :white_check_mark: 表示该版本**受到支持**,我们会**积极处理**有关此版本的安全问题。
- :x: 表示该版本已被标记为**过时版本**,我们**不会受理**此版本的安全问题。
> 若您的版本未受到支持,不妨考虑更新到最新受支持的版本再尝试复现此安全问题。

| 版本 | 是否支持          |
| ------- | ------------------ |
| v1.1_20250501   | :white_check_mark: |
| v1.1_20250412   | :white_check_mark: |
| v1.1_20250216   | :x:                |
| v1.0开头的版本   | :x: |
